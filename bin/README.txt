tested on linux machines in g7

To compile:
cd p1-gbonann1/src
javac rc6/*.java

To run:
cd p1-gbonann1/src
java rc6.Driver input.txt output.txt

OUTPUT FILE MUST ALREADY EXIST IN ORDER TO RUN. 
(Create a blank *.txt file in src folder)

if a hex character is a single digit it will not print the 0 before it.
ie: '02' appears as '2'.
